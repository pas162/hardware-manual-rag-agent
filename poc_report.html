The LLM (e.g. Claude) drives the conversation and decides which tool to call. We expose tools via MCP — the agent consumes them without any modification on our side.
Claude Desktop Cursor Any MCP-enabled client Calls tools mid-conversation Returns cited answers
MCP Server Shipped
Model Context Protocol is an open standard for tool-calling. The server is ~100 lines of Python (FastMCP) and exposes three tools the agent can call at any point during a chat.
search_um
Semantic search over all manual content. Returns ranked chunks with citations.
register_lookup
Exact lookup for register addresses, reset values, and bit fields, read directly from the manufacturer's register database.
get_figure
Fetches a diagram by ID and returns the actual image for the LLM to see.
Database Both live
Two data paths, each suited to its job: ChromaDB finds passages by meaning; the Smart Manual database returns exact register values that must never be guessed.
ChromaDBvector · meaning search · prose & figure chunks Smart Manual DBexact register & bit-field lookup, queried live Runs 100% locally · no cloud dependency
